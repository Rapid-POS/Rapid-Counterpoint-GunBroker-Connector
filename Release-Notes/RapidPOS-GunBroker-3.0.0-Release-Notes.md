# Rapid POS GunBroker Integration Release Notes  
_Release Date: August 1, 2025_

---

## Overview  
This release includes new enhancements to the GunBroker integration, improving deployment processes, compatibility, listing functionality, fee tracking, and configuration options.

---

## New Features & Enhancements  

### GunBroker CI/CD Update  
The GunBroker integration is now part of our CI/CD pipeline, enabling faster, more reliable testing and deployment of future updates.

### Updated API Request Headers  
Updated API request headers to align with GunBroker's new requirements, ensuring continued compatibility with GunBroker's platform.

### Default GunBroker Tax Code  
Added the ability to set and change the default GunBroker Tax Code used in Counterpoint.

### GunBroker Listing & Relisting Support  
Added support for creating listings directly in GunBroker from Counterpoint and leveraging GunBroker's relisting functionality.  
- Counterpoint will continue to automatically pull completed GunBroker orders.

### GunBroker Fee Import  
Added the ability to import GunBroker fees into Counterpoint for more accurate profitability reporting.

### Serial Number Listing Option  
Added the option to list firearms with or without serial numbers.
